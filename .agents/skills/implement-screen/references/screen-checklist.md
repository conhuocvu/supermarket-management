# Screen Checklist

## Layout

- [ ] Uses Scaffold or AppScaffold
- [ ] Responsive on phone and tablet
- [ ] No overflow
- [ ] Proper padding

## Theme

- [ ] Uses ThemeData
- [ ] No hardcoded colors
- [ ] Typography from textTheme

## Components

- [ ] Uses AppButton
- [ ] Uses AppCard
- [ ] Uses AppTextField
- [ ] Uses StatusChip

## States

- [ ] Loading
- [ ] Empty
- [ ] Error
- [ ] Success

## Performance

- [ ] Uses const where possible
- [ ] Uses ListView.builder
- [ ] Extract reusable widgets

## Accessibility

- [ ] Touch targets ≥ 48dp
- [ ] Icons have semantic meaning
- [ ] Good contrast

## Code Quality

- [ ] No duplicated widgets
- [ ] No dead code
- [ ] Formatted with dart format
